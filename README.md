# MDR-AI

Telegram-first AI video generation platform.

MDR-AI allows users to create AI-generated videos directly inside Telegram using text prompts, multi-scene workflows, automatic video assembly, and modern generative video models.

The platform is designed for creators, marketers, businesses, agencies, meme creators, social media teams, and anyone who wants to produce short-form AI video content without learning complex editing software.

---

# Overview

MDR-AI transforms natural language prompts into complete AI-generated videos.

Users interact with a Telegram bot, describe an idea in plain text, choose an AI model, generate multiple scenes, regenerate individual shots if necessary, and receive a ready-to-publish video directly inside Telegram.

The platform removes the need for:
- traditional video editing software
- complicated AI interfaces
- local GPU hardware
- prompt engineering knowledge
- multi-tool workflows

MDR-AI combines AI video generation, scene orchestration, automatic assembly, and publishing workflows into a single Telegram-native experience.

---

# Core Features

## Text-to-Video Generation

Generate videos from simple prompts.

Example:

The system sends prompts to selected AI video generation models and returns generated scenes directly inside Telegram.

# How It Works

## Step 1 — Send a Prompt

Users describe a video idea in natural language.

Examples:
- "Luxury perfume commercial in slow motion"
- "Cute cat cooking pasta anime style"
- "Sci-fi spaceship landing on Mars"
- "Minimalist coffee ad for TikTok"

The prompt can include:
- style
- mood
- camera movement
- lighting
- pacing
- environment
- aspect ratio
- number of scenes

---

## Step 2 — Generate Scenes

MDR-AI creates scenes using selected AI generation models.

Each scene can be:
- accepted
- regenerated
- modified
- replaced

This iterative workflow gives users more control over final outputs.

---

## Step 3 — Assemble Video

Accepted scenes are automatically merged into a final composition.

The system may additionally generate:
- title
- cover image
- social caption
- publishing description

The final result is delivered directly inside Telegram.
